SENSORS (circles on map)

      
Satellite — Wide-area, long range. Best for strategic targets.

      
Drone — Tactical, precise, short range. Best for kinetic targets.

      
Radar — Fast detection. Best for airspace targets.

      
THREATS (triangles on map)

      
HIGH (P3) — Critical. Cover immediately. +2.0 reward if covered.

      
MED (P2) — Moderate risk. +1.0 reward.

      
LOW (P1) — Minor risk. +0.5 reward.

      
AGENTS (multi-agent mode only)

      
Satellite Agent — Controls satellite sensors. Sees partial map. Specialises in strategic targets.

      
Drone Agent — Controls drone sensors. Sees partial map. Specialises in kinetic targets.

      
Radar Agent — Controls radar sensors. Sees partial map. Specialises in airspace targets.

      
Command Agent — Sees ALL sensors & targets. No fixed sensor. Override authority — resolves ties. Costs −0.5 per override.

      
ARCS (colored lines)

      
Each colored line = one agent's proposal. Color matches the agent. Disappears after 4 seconds.
